# AI Workshop - OpenEdge ABL Project

This repository contains an OpenEdge ABL project demonstrating business entity patterns and UI development.

## Project Structure

- `src/` - Source code (business logic and UI windows)
- `src/business/` - Business entity classes and datasets
- `dump/` - Database schema files
- `doc/` - Documentation
- `.windsurf/` - Windsurf IDE configuration and workflows

## Features

- Business Entity Pattern implementation
- Customer and Item management windows
- Sports2000 database integration
- OpenEdge ABL best practices

## Getting Started

1. Ensure OpenEdge is installed and configured
2. Review the database schema in `dump/sports2000.df`
3. Build the project using `build.xml`
4. Run the UI windows from `src/`
